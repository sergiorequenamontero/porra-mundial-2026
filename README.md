# Porra Mundial 2026

Web estática para GitHub Pages.

## Archivos

- `index.html`: la web.
- `data.json`: participantes, partidos, pronósticos y resultados.

## Contraseña de administrador

La contraseña provisional es:

`admin2026`

## Cómo publicarlo gratis en GitHub Pages

1. Crea una cuenta en GitHub si no tienes.
2. Crea un repositorio nuevo llamado, por ejemplo: `porra-mundial-2026`.
3. Sube `index.html` y `data.json`.
4. Entra en `Settings` > `Pages`.
5. En `Build and deployment`, elige:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Guarda.
7. GitHub te dará una URL pública.

## Cómo actualizar resultados

1. Entra en la web.
2. Pulsa `Modo admin`.
3. Introduce `admin2026`.
4. Cambia resultados o pronósticos.
5. Pulsa `Descargar data.json actualizado`.
6. Sustituye el `data.json` del repositorio por el nuevo.

Importante: si solo editas desde la web pero no subes el nuevo `data.json` a GitHub, los cambios solo se guardan en tu navegador.
